# AliceGlowStore

Sistema de gestão (Vite + React + TypeScript + Tailwind + shadcn/ui).

## Rodar localmente

Pré-requisitos: Node.js (recomendado LTS).

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Configuração de API

O proxy do Vite está configurado em [vite.config.ts](vite.config.ts) para encaminhar `/api` para o backend.
